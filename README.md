# Spatial Periphery Plots

## Introduction
Geospatial data, also known as spatial data is data describing information about a location on the Earth. Geometric data is one type of spatial data that presents information on a 2-dimensional surface. Geographic data presents information on a
sphere. Spatial data has applications in various domains such as weather, ecology, urban planning, politics, agriculture, and transportation. Spatial data is hard to interpret and thus, it is essential to develop novel ideas to understand the data and answer relevant questions.

## Problem Statement
Periphery plots is a relatively newer approach to visualize temporal data. This project aims to apply principles of temporal periphery plots to spatial domains to visualize
spatial data. The Focus Area details the information in a given area. The Context Area which is on the Periphery of the Focus Area summarizes the information around the Focus Area.


## Setup Instructions

- Clone this repository.
- Open `index.html` in your web browser.

## Assumptions

- Input data is in CSV format.
- First column: X coordinate, Second column: Y coordinate.
- Additional columns: Value columns (numerical or categorical).
- Data file is located in the `data/` folder.
